Ano ang HTML?

Ang HTML (Hyper Text Markup Language) ay isang web-based na wika na nagsasabi sa iyong browser kung paano ipakita ang text at mga larawan sa isang webpage. Gumagamit ka ng mga tag upang ipahiwatig ang iba't ibang elemento, matukoy ang pagkakalagay ng mga talata, heading, magdagdag ng mga larawan at video.

Ano ang CSS?

Ang CSS o Cascading Style Sheets ay nagdaragdag ng styling gaya ng mga font, kulay o layout sa iyong website na ginagawang mas aesthetically pleasing ang website.
