CSS

Ang CSS ay ang wikang ginagamit namin sa pag-istilo ng isang Web page.

Ano nga ba ang CSS?

Ang Cascading Style Sheets ay isang style sheet na wika na ginagamit para sa paglalarawan ng presentasyon ng isang dokumentong nakasulat sa isang markup language gaya ng HTML o XML. Ang CSS ay isang pundasyong teknolohiya ng World Wide Web, kasama ng HTML at JavaScript.

CSS Syntax

Ang isang panuntunan ng CSS ay binubuo ng isang tagapili at isang bloke ng deklarasyon.

Itinuturo ng selector ang HTML element na gusto mong i-istilo.
Ang bloke ng deklarasyon ay naglalaman ng isa o higit pang mga deklarasyon na pinaghihiwalay ng mga semicolon.
Kasama sa bawat deklarasyon ang isang CSS property name at isang value, na pinaghihiwalay ng colon.
Ang maramihang mga deklarasyon ng CSS ay pinaghihiwalay ng mga semicolon, at ang mga bloke ng deklarasyon ay napapalibutan ng mga kulot na brace.

Halimbawa

p { color: red; text-align: center; }
  • Ang p ay isang tagapili o selector sa CSS (ito ay tumuturo sa HTML na elemento na gusto mong i-istilo).
  • ang kulay ay isang pag-aari, at ang pula ay ang halaga ng pag-aari
  • ang text-align ay isang property, at ang center ay ang property value

Basics

Syntax

selector{ property: value; property2: value2; }

Include external css file

<link rel="stylesheet" type="text/css" href="/style.css"/>

Internal styles

<style type="text/css">
div { color: #444;}
</style>

Inline styles

<tag style="property: value"> </tag>
