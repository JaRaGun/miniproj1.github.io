HTML

Ang HTML (HyperText Markup Language) ay ang code na ginagamit upang buuin ang isang web page at ang nilalaman nito. Halimbawa, ang nilalaman ay maaaring ibalangkas sa loob ng isang hanay ng mga talata, isang listahan ng mga naka-bullet na punto, o paggamit ng mga larawan at talahanayan ng data. Gaya ng iminumungkahi ng pamagat, ang artikulong ito ay magbibigay sa iyo ng pangunahing pag-unawa sa HTML at sa mga function nito.

Ano nga ba ang HTML?

Ang HTML ay isang markup language na tumutukoy sa istruktura ng nilalaman. Ang HTML ay isang hanay ng mga elemento na ginagamit upang i-wrap o ilakip ang iba't ibang nilalaman upang ipakita o kumilos sa isang partikular na paraan. Halimbawa:
<tagname> Content goes here... </tagname>

HTML Headings

Ang mga heading ng HTML ay mga pamagat o subtitle na gusto mong ipakita sa isang webpage.
Ang mga HTML heading ay tinukoy gamit ang <h1> hanggang <h6> na mga tag.
Tinutukoy ng <h1> ang pinakamahalagang heading.
Tinutukoy ng <h6> ang hindi gaanong mahalagang heading.

# Sample
1 <h1>Hello panget</h1>
2 <h2>Hello panget</h2>
3 <h3>Hello panget</h3>
4 <h4>Hello panget</h4>
5 <h5>Hello panget</h5>
6 <h6>Hello panget</h6>

Mahalaga ang Mga Pamagat o headings

Ginagamit ng mga search engine ang mga heading upang i-index ang istraktura at nilalaman ng iyong mga web page.
Ang mga gumagamit ay madalas na nag-skim ng isang pahina sa pamamagitan ng mga heading nito. Mahalagang gumamit ng mga heading upang ipakita ang istraktura ng dokumento.
Dapat gamitin ang <h1> heading para sa mga pangunahing heading, na sinusundan ng <h2> heading, pagkatapos ay ang hindi gaanong mahalaga <h3>, at iba pa.

HTML Paragraphs

Ang HTML <p> na elemento ay tumutukoy sa isang talata. Palaging nagsisimula ang isang talata sa isang bagong linya, at awtomatikong nagdaragdag ang mga browser ng ilang puting espasyo (isang margin) bago at pagkatapos ng isang talata.

# Sample
1 <p>Ako ay may lobo lumipad sa langit</p>

HTML DISPLAY

Hindi ka makatitiyak kung paano ipapakita ang HTML.
Malalaki o maliliit na screen, at na-resize na mga bintana ay lilikha ng iba't ibang resulta.
Sa HTML, hindi mo mababago ang display sa pamamagitan ng pagdaragdag ng mga dagdag na espasyo o dagdag na linya sa iyong HTML code.
Awtomatikong aalisin ng browser ang anumang mga karagdagang espasyo at linya kapag ipinakita ang pahina.

Subukan mo itong ilagay sa compiler:

naglalaman ng maraming linya
sa source code,
ngunit ang browser
binabalewala ito. </p>
<p>
